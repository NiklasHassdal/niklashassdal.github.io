EasyView

I'm the lead developer of EasyView at Intab Interface-Teknik AB. EasyView is a desktop application used for plotting graphs, analyzing data and performing calculations. It has built in support for configuring and reading data from data loggers such as the PC-logger, Gemini Tinytag and the Novus FieldLogger but you can also read data from other sources such as Modbus RTU, Modbus TCP or OPC UA. It's also possible to import data from Excel or CSV files.

IntabCloud.com

IntabCloud.com, formely known as ComfortLog.com or Kyrklog.se, is a cloud service that a multitude of data loggers can send data to. Features include viewing graphs, alarms (E-mail and SMS), exporting data, automatic PDF reports and many more.
